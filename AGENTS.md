# AGENTS.md

## GWD naming rule

When editing GWD skill files, docs, commands, examples, or templates, avoid direct references to `GTD`, `Getting Things Done`, or `ZCode` in user-facing text.

Use GWD-native language instead:

- `GWD`
- `metodo GWD`
- `workflow GWD`
- `horizontes de foco`
- `capturar`, `esclarecer`, `organizar`, `revisar`, `executar`

Allowed exception: internal implementation notes may mention tool paths or platform mechanics only when required for correctness, but public-facing copy should keep GWD as the primary method/brand.
